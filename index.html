Comandos Basicos de la terminal | Class #1

  • Listar archivos: ls
  • Listar archivos para ver su peso de una manera más legible: ls -lh
  • Listar archivos ocultos: ls -a
  • Identificar la ruta en la que estamos en nuestro sistema: pwd
  • Movernos entre directorios: cd
  • Limpiar la terminal: clear
  • Información acerca del tipo de elemento: file

    Comandos Basicos de la terminal | Class #2

  • Listar archivos ocultos: ls -a
  • Listar archivos incluyendo los ocultos: ls -la
  • Listar archivos por tamaño: ls -lS
  • Listar archivos decendente (z-a): ls -lr
  • Listar archivos por ramas: tree
  • Listar archivos por ramas en niveles: tree -L (1 | 2 | 3)
  • Crear directorios: mkdir (nombre-del-directorio)
  • Crear archivos: touch (nombre-del-archivo)
  • Copiar archivos: cp (nombre-del-archivo) (nombre-de-la-copia-del-archivo)
  • Mover archivos: mv (nombre-del-archivo) (nombre al directorio que deseas mover)
  • Renombrar archivos: mv (nombre-actual) (nuevo-nombre)
  • Borrar archivos: rm (nombre-del-archivo)
  • Borrar archivos interactivamente: rm -i (nombre-del-archivo)
  • Borrar directorios: rm -ri (nombre-del-directorio)
