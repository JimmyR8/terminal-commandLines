Comandos Basicos de la terminal | Class #1

  • Listar archivos: ls
  • Listar archivos para ver su peso de una manera más legible: ls -lh
  • Listar archivos ocultos: ls -a
  • Identificar la ruta en la que estamos en nuestro sistema: pwd
  • Movernos entre directorios: cd
  • Limpiar la terminal: clear
  • Información acerca del tipo de elemento: file

    Comandos Basicos de la terminal | Class #2

  • Listar archivos ocultos: ls -a
  • Listar archivos incluyendo los ocultos: ls -la
  • Listar archivos por tamaño: ls -lS
  • Listar archivos decendente (z-a): ls -lr
  • Listar archivos por ramas: tree
  • Listar archivos por ramas en niveles: tree -L (1 | 2 | 3)
  • Crear directorios: mkdir (nombre-del-directorio)
  • Crear archivos: touch (nombre-del-archivo)
  • Copiar archivos: cp (nombre-del-archivo) (nombre-de-la-copia-del-archivo)
  • Mover archivos: mv (nombre-del-archivo) (nombre al directorio que deseas mover)
  • Renombrar archivos: mv (nombre-actual) (nuevo-nombre)
  • Borrar archivos: rm (nombre-del-archivo)
  • Borrar archivos interactivamente: rm -i (nombre-del-archivo)
  • Borrar directorios: rm -ri (nombre-del-directorio)

    Comandos Basicos de la terminal | Class #3

  • Mostrar las 10 primeras lineas de un archivo de texto: head (nombre-del-archivo)
  • Mostrar el numero de lineas que le asigne: head (nombre-del-archivo) -n (numero)
  • Mostrar las ultimas 10 lineas de un archivo de texto: trail (nombre-del-archivo)
  • Mostrar las ultimas lineas que le asigne: trail (nombre-del-archivo) -n (numero)
  • Observar todo el contenido de un archivo de texto: less (nombre-del-archivo)
  • Abrir en un programa el archivo: xdg-open (nombre-del-archivo) "usando el sistema operativo Linux"
  • Abrir una carpeta: xdg-nautilus (nombre-del-directorio) "usando el sistema operativo Linux"

    Comandos Basicos de la terminal | Class #4

  • Saber que comando estoy utilizando: type (comando)
  • Crear alias de comandos: alias (una-letra)="(comando) -lh"
  • Pedir ayuda para los comandos: help (comando) "con shell"
  • Pedir ayuda de los comandos: (comando) --help "con el mismo comando pido la ayuda sin nesecidad de shell"
  • Manual de usuario de un comando: man (comando)
  • Manual de usuario de un comando de manera resumida: infor (comando)
  • Información muy abreviada de un comando: whatis (comando)

    Comandos Basicos de la terminal | Class #5

  • Buscar toda la información dentro de una carpeta: ls *
  • Buscar toda la información dentro de una carpeta especificamente: ls *.(tipo-de-archivo)
  • Buscar toda la información dentro de una carpeta con un nombre especificamente y varios caracteres despues: ls (nombre-del-archivo)*
  • Buscar toda la informacion dentro de una carpeta con un nombre especificamente con el numero caracteres que le asignemos: ls (nombre-del-archivo)?
  • Buscar y obtener toda la informacion de dirtectorios y archivos que empiencen con un letra mayuscula: ls -d [[:upper:]]*
  • Buscar y obtener toda la informacion de dirtectorios y archivos que empiencen con un letra miniscula: ls -d [[:lower:]]*
  • Buscar toda la informacion de archivos y directorios por caracteres asignados: ls [(letra-para-la-busqueda)]*

    Comandos Basicos de la terminal | Class #6

  • Redirige la salida de un comando a un archivo. El mismo sobrescribe el contenido del archivo a donde se redirige la salida: ls (nombre-del-archivo-o-diretorio) > (nombre-del-archivo-a-crear).txt
  • Redirige el input de un comando hacia un archivo.: ls (nombre-del-archivo-o-diretorio) < (nombre-del-archivo-a-crear).txt
  • Concatena la salida de un comando a un archivo, si no existe el archivo lo crea: ls (nombre-del-archivo-o-diretorio) >> (nombre-del-archivo).txt
  • Redirige la salida de error de un comando a un archivo: ls (nombre-del-archivo-o-diretorio) 2> (nombre-del-archivo).txt
  • Redirige la salida de un comando, que se ejecuto satisfactoriamente o un comando que presento errores, a un archivo: ls (nombre-del-archivo-o-diretorio) > (nombre-del-archivo) 2>&1

    Comandos Basicos de la terminal | Class #7

  • Genera un stdout con el texto que tenemos: echo (texto)
  • Muestra los dos archivos concatenados: cat (nombre-del-archivo) (nombre-del-archivo)
  • El pipe operator "|" hace que el stdout de un comando sea el stdin de otro comando: ls -lh | less (nombre-del-archivo)
  • "tee" hace algo parecido a "">"", pero dentro de los pipes: ls -lh | tee (nombre-del-archivo) | less
  • Generar texto con bonitos colores: echo "texto" | lolcat
  • Generar una vaquita con un mensaje de texto: cowsay "texto"
  • Generar una vaquita con un mensaje de texto con color, usando el pipe operator: cowsay "texto" | lolcat
