    Cómo se manejan los permisos.

    Cuando listamos con ls -l se muestran varias cosas. Los tipos de archivos:

  • - archivo normal.
  • d directorio.
  • l link simbólico.
  • b archivo de bloque especial.

    • Tipos de modos:

    rwx corresponde con read, write y execute. Se representan con 3 bits, y los podemos manejar a través de un modo octal, esto es, pasar de binario a número.

  • rwx (1,1,1) dueño. En modo octal es 7.
  • r-x (1,1,1) grupo. En modo octal es 5.
  • r-x (1,0,1) world. Octal 5.

    • Modo simbólico:

    Esto es para asignar los permisos a los diferentes posibles usuarios.

  • u Solo para el usuario.
  • g Solo para el grupo.
  • o Solo para otros (world).
  • a Aplica para todos.

  • .tar se usa mucho en repositorios. Para comprimir archivos: tar -cvf (nombre-del-archivo).tar (nombre-de-la-carpeta-a-comprimir) donde c → compress, v → verbose, f→ file.
  • .gz es un poco mejor, el mismo comando pero con la bandera z→zip: tar -cvzf (nombre-del-archivo).gz (nombre-de-la-carpeta-a-comprimir) El algoritmo gzip es muy eficiente para comprimir.
  • Para descomprimir es el mismo comando pero con la bandera x → decompress en lugar de c → compress. Para que funcione, debemos descomprimir usando el mismo tipo de compresión (tar o zip)
  • .zip es uno muy común: zip -r (nombre-del-archivo).zip (nombre-del-directorio) Se comprime pero de forma recursiva con -r
  • Para descomprimir en zip usamos: unzip (nombre-del-archivo).zip
  • Esta tambien el rar se usa igual que el zip, para comprmir es: rar -r (nombre-del-archivo).rar (nombre-del-directorio) Se comprime pero de forma recursiva pero con -r
  • Para descomprimir en rar usamos: unrar (nombre-del-archivo).rar

    Una de las utilidades más importantes de la terminal es el editor de texto. Hay diferentes opciones, pero Vim es uno de los mas sencillos y populares.

  • vi es la versión vieja de editor de texto: vi (nombre-del-archivo)
  • vim es la version moderna o pues (Actualizada): vim (nombre-del-archivo)
  • Al igual que con less para buscar una palabra, podemos hacerlo en Vim con: /(palabra) Te lleva a la primera coincidencia.
  • Para eliminar una línea, desde el modo normal (si estamos en modo INSERT solo presionamos Esc y listo), nos ponemos al inicio de la línea y presionamos la tecla d dos veces.
  • Para guardar nos ubicamos en el modo NORMAL, presionamos / y escribimos la letra w, Seria asi: /w
  • Para guardar y salir del editor de texto nos ubicamos en el modo NORMAL, presionamos / y escribimos las letras wq
  • Para salir de forma forzada por alguna razon que no te deje seria con ! ejemplo: /wq!
